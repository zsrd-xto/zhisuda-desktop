# 职速达隐私政策

> **生效日期**: 2026-06-24 | **版本**: v1.0 | **产品**: 职速达桌面应用

职速达（以下简称「我们」）重视您的隐私。本政策说明我们如何收集、使用、存储和保护您的信息。

---

## 一、适用范围

本政策适用于职速达桌面应用程序及官方网站（zhishuda.com）。使用本软件即表示您同意本政策。

---

## 二、我们收集的信息

### 2.1 MVP（v0.1）本地模式 — 最小化收集

在 MVP 内测阶段，本软件 **无需注册账号**，原则上 **不向我们的服务器上传您的个人求职数据**。

本地存储的信息包括：

| 数据类型 | 存储位置 | 用途 |
|----------|----------|------|
| 设备标识（UUID） | 本地 SQLite | 区分本地用户 |
| 昵称（可选） | 本地 SQLite | 个性化显示 |
| 简历内容与文件 | 本地磁盘 + SQLite | 求职与投递 |
| 求职偏好 | 本地 SQLite | 岗位匹配 |
| 招聘平台 Cookie | 本地加密存储 | 维持平台登录态 |
| 投递记录 | 本地 SQLite | 数据看板 |
| 操作日志 | 本地 SQLite | 故障排查 |

### 2.2 官网访问

访问官网时，托管服务商（如 Vercel/CDN）可能记录标准访问日志（IP、浏览器类型、访问时间），用于安全与统计。我们不使用第三方广告追踪。

### 2.3 崩溃上报（可选）

若您主动开启「发送崩溃报告」：
- 我们可能收集匿名崩溃堆栈、应用版本、操作系统版本；
- **不包含** 简历内容、Cookie 或招聘平台聊天记录。

默认 **关闭**，需您明确同意后方才启用。

### 2.4 未来版本（v0.2+）

引入手机号注册后，我们将收集：
- 手机号码（用于登录验证）
- 订阅状态（用于付费功能鉴权）

届时将更新本政策并征得您的同意。**招聘平台 Cookie 与简历内容仍默认仅存本地，不上传云端。**

---

## 三、我们不收集的信息

- 招聘平台（Boss 直聘等）的 **账号密码**
- 未经您授权的聊天记录全文上传
- 您的 API Key 内容（v0.2 AI 功能仅在本地使用，直连模型提供商）

---

## 四、信息使用目的

| 目的 | 说明 |
|------|------|
| 提供核心功能 | 简历管理、岗位匹配、辅助投递 |
| 改善产品 | 匿名崩溃分析（opt-in） |
| 版本更新 | electron-updater 检查新版本 |
| 客户服务 | 您主动联系我们时的沟通记录 |

---

## 五、信息存储与安全

5.1 **本地优先**：MVP 阶段您的求职数据主要存储在您的设备上。

5.2 **加密措施**：
- 招聘平台 Cookie：AES-256 加密后本地存储
- API Key（v0.2）：AES-256 加密后本地存储

5.3 **保留期限**：数据在您主动清除或卸载软件前一直保留；清除后不可恢复。

---

## 六、信息共享与披露

6.1 我们不会出售您的个人信息。

6.2 以下情形除外：
- 获得您的明确同意
- 法律法规要求或司法机关依法要求
- 为保护我们或用户的合法权益所必需

6.3 AI 功能（v0.2+）：当您使用 AI 回复或简历优化时，相关内容将 **直接从您的设备发送至您配置的模型提供商**（如 OpenAI、通义千问），不经过职速达服务器中转。

---

## 七、您的权利

您有权：
- **查阅** 本地存储的数据（通过应用界面）
- **更正** 简历与偏好信息
- **删除** 全部本地数据（设置 → 清除所有数据）
- **拒绝** 崩溃上报（默认已拒绝）
- **卸载** 应用以停止数据处理

---

## 八、未成年人保护

本软件面向具有完全民事行为能力的求职者。未满 18 周岁请在监护人指导下使用。

---

## 九、政策更新

我们可能适时修订本政策。重大变更将通过官网或应用内通知。继续使用即视为接受更新后的政策。
